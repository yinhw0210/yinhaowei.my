# 网站图标说明

## 设计理念

所有图标都采用90年代Windows 95复古风格设计，与网站整体主题保持一致。

## 图标文件

### 1. favicon.svg (16x16)
- **用途**: 浏览器标签栏图标
- **设计**: 迷你Windows 95窗口，带有蓝色标题栏和黑色终端内容
- **特点**: 绿色终端提示符，模拟DOS/命令行界面

### 2. icon.svg (32x32)
- **用途**: 通用图标，更大尺寸显示
- **设计**: 完整的Windows 95窗口框架
- **元素**:
  - 灰色窗口边框（#C0C0C0）
  - 蓝色渐变标题栏（#000080 → #1084D0）
  - 黑色终端内容区
  - 绿色命令行提示符（#00FF00）

### 3. apple-icon.svg (180x180)
- **用途**: iOS设备"添加到主屏幕"图标
- **设计**: 高分辨率Windows 95窗口
- **优化**: 适配iOS圆角显示

### 4. manifest.json
- **用途**: PWA（渐进式Web应用）配置
- **主题色**: 
  - 背景色: #008080 (Windows 95 Teal)
  - 主题色: #000080 (Windows 95 Blue)

## 设计元素

### 颜色方案
- **窗口灰**: #C0C0C0 (经典Windows 95灰色)
- **标题栏蓝**: #000080 → #1084D0 (渐变)
- **终端黑**: #000000
- **终端绿**: #00FF00 (经典DOS绿色)
- **高光白**: #FFFFFF
- **阴影黑**: #000000

### 视觉特征
- 3D凸起边框效果
- 像素完美的几何形状
- 无圆角设计（符合90年代审美）
- 高对比度配色
- 怀旧的终端命令行界面

## 技术规格

- **格式**: SVG (矢量图形，可无损缩放)
- **兼容性**: 支持所有现代浏览器
- **备用方案**: 保留原有favicon.ico作为后备
- **响应式**: 自适应不同设备和分辨率

## 使用说明

图标会自动在以下场景显示：
- 浏览器标签页
- 书签栏
- iOS/Android主屏幕
- Windows任务栏
- macOS Dock

无需额外配置，Next.js会自动处理图标的加载和显示。
